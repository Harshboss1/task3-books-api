# Books REST API

This is a simple REST API built using Node.js and Express to manage books.

## Features
- GET /books → Get all books
- POST /books → Add book
- PUT /books/:id → Update book
- DELETE /books/:id → Delete book

## Tech Used
- Node.js
- Express.js

## Run Project
1. Install dependencies:
   npm install

2. Start server:
   node index.js

3. Open:
   http://localhost:3000/books
